# UI Agent Integration Notes

This package provides a unified Cursor setup for UI and design-system work.

## Included subagents

- `design-system-guardian`
- `component-architect`
- `ui-refactor-specialist`
- `design-token-auditor`
- `responsive-accessibility-auditor`
- `ui-stack-adapter`

## Included commands

- `/adopt-design-system`
- `/design-audit`
- `/ui-refactor`
- `/ui-ship-check`

## Included rules

- `ui-governance.mdc`
- `ui-adoption-first.mdc`

## Recommended first run

1. Add the files to the repository.
2. Run `/adopt-design-system`.
3. Review generated and updated docs.
4. Use `/design-audit` before broader cleanup.
5. Use `/ui-refactor` for targeted normalization.
