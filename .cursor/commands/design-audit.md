# /design-audit

Run a design-system and UI consistency audit.

## Required workflow
1. read `docs/ai/design-system.md`, `docs/ai/ui-stack.md`, `docs/ai/ui-patterns.md`
2. use `design-system-guardian`
3. use `design-token-auditor` when tokens or utility classes are involved
4. use `responsive-accessibility-auditor` for user-facing flows

## Output
Report:
- inconsistencies found
- recommended fixes
- docs to update
